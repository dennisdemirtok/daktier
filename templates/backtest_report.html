📚 Metod

  • PIT-data: Alla KPI hämtade från Borsdata API med period_year < year-filter (ingen look-ahead).
  • Universum: top-N market cap per land per år (dynamisk).
  • Forward return: close-price 12m efter screening-datum (15 juli varje år).
  • Survivorship bias: ~2-4% överskattning, ej justerat. Robusthet kommer främst från sub-period-test.
  • Sub-period: Early (2015-2019) vs late (2020-2024) för att fånga regim-bias.

Backtest-data uppdateras vid större metodförändringar.
